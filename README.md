->Freelance Hiring Platform

GigFlow is a full-stack freelancing platform where clients can post gigs, freelancers can place bids, and clients can hire freelancers.

 ->Features
- User authentication (Register/Login)
- Create & browse gigs
- Search gigs by title
- Place bids on gigs
- Gig owner can view all bids
- Hire a freelancer
- Protected routes using JWT


Tech Stack
->Frontend
- React.js
- Tailwind CSS
- Axios
- React Router
- Context API

->Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
